# Symfony features lab (with Docker) 

:test_tube: A playground to test the new Symfony 6+ features in real conditions!

## Dev Env
- `Symfony 6.1`
- `PHP 8.1`
- `Docker support`
- `PHPUnit 9.5`
- `PHPStan 1.8`
- `Symfony Panther 2.0`
- `Behat 3.11`
- `PHP CS Fixer 3`
